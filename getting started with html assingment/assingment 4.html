Short note on tag and element

HTML Tags

If you want to build a beautiful website, tags are essential elements that help you achieve that. An HTML tag acts as a container for content or other HTML tags. Tags are words enclosed within < and > angle brackets. They serve as keywords that instruct the web browser on how to format and display the content

Lorem ipsum dolor sit amet.

closing tag content is Lorem ipsum dolor sit amet

HTML Elements

An HTML element is a complete set that consists of a start tag (or opening tag), content, and an end tag (or closing tag). HTML Element = Start Tag + Content + End Tag
